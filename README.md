### Requirements

- Java (v17)
- Maven (v3.2.3)
- Docker (PostgreSQL database)

### Instalattion

1. Clone the repository.<br>
   `git clone https://github.com/gabrielpenteado/api-java-springboot-docker`

2. Create a PostgreSQL database Docker container.<br>
   `docker compose up -d`

3. Run Application.<br>
   `mvn spring-boot:run`
